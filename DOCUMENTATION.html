Modding API Documentation

The Modding API provides a set of functions that allow you to extend and modify the functionality of your game through mods. This documentation outlines the available functions and their usage.

Getting Started

To use the Modding API, follow these steps:

  1. Create a ModLoader object in your game scene.
  2. Place your mod scripts in a folder named Mods within your project's Assets folder.
  3. Each mod script should be written in Lua and have a .lua file extension.

API Functions

1. SpawnGameObject

public void SpawnGameObject(GameObject prefab, Vector3 position)

Spawns a game object at the specified position.

  • prefab: The prefab of the game object to spawn.
  • position: The position in the world where the game object should be spawned.

Example:

ModLoader.Instance.SpawnGameObject(Resources.Load("CubePrefab"), Vector3.zero);

2. DeleteGameObject

public void DeleteGameObject(GameObject gameObject)

Deletes a game object from the scene.

  • gameObject: The game object to delete.

Example:

GameObject obj = GameObject.FindWithTag("ObjectToDelete");
if (obj != null)
{
    ModLoader.Instance.DeleteGameObject(obj);
}

3. AddUiCanvas

public Canvas AddUiCanvas(string canvasName)

Adds a UI canvas to the scene.

  • canvasName: The name of the canvas.

Example:

Canvas canvas = ModLoader.Instance.AddUiCanvas("UICanvas");

4. AddUiText

public Text AddUiText(Canvas canvas, string textContent)

Adds a UI text element to a canvas.

  • canvas: The canvas to which the text element should be added.
  • textContent: The content of the text element.

Example:

Text textElement = ModLoader.Instance.AddUiText(canvas, "Hello, World!");

5. AddUiButton

public Button AddUiButton(Canvas canvas, string buttonLabel)

Adds a UI button element to a canvas.

  • canvas: The canvas to which the button element should be added.
  • buttonLabel: The label text of the button.

Example:

Button buttonElement = ModLoader.Instance.AddUiButton(canvas, "Click Me!");

6. AddDropdown

public Dropdown AddDropdown(Canvas canvas, string dropdownName, string[] options, Action<int> onSelectionChanged)

Adds a UI dropdown element to a canvas.

  • canvas: The canvas to which the dropdown element should be added.
  • dropdownName: The name of the dropdown element.
  • options: An array of strings representing the dropdown options.
  • onSelectionChanged: A callback function to be invoked when the dropdown selection changes. The function should take an integer parameter representing the selected index.

Example:

string[] options = { "Option 1", "Option 2", "Option 3" };
Action<int> onSelectionChanged = index => Debug.Log("Selected option: " + options[index]);
Dropdown dropdownElement = ModLoader.Instance.AddDropdown(canvas, "Dropdown", options, onSelectionChanged);

7. AddLight

public Light AddLight(Vector3 position, Color color)

Adds a light source to the scene.

  • position: The position in the world where the light should be placed.
  • color: The color of the light.

Example:

Color lightColor = new Color(1f, 1f, 0f);
Light light = ModLoader.Instance.AddLight(Vector3.zero, lightColor);

8. ChangeTag

public void ChangeTag(GameObject gameObject, string newTag)

Changes the tag of a game object.

  • gameObject: The game object whose tag should be changed.
  • newTag: The new tag value.

Example:

GameObject obj = GameObject.FindWithTag("OldTag");
if (obj != null)
{
    ModLoader.Instance.ChangeTag(obj, "NewTag");
}

9. ChangeLayer

public void ChangeLayer(GameObject gameObject, int newLayer)

Changes the layer of a game object.

  • gameObject: The game object whose layer should be changed.
  • newLayer: The new layer value.

Example:

GameObject obj = GameObject.FindWithTag("Object");
if (obj != null)
{
    int newLayer = LayerMask.NameToLayer("NewLayer");
    ModLoader.Instance.ChangeLayer(obj, newLayer);
}

10. AddTag

public void AddTag(string tagName)

Adds a new tag to the game object.

  • tagName: The name of the new tag.

Example:

ModLoader.Instance.AddTag("NewTag");

11. AddLayer

public void AddLayer(string layerName)

Adds a new layer to the game object.

  • layerName: The name of the new layer.

Example:

ModLoader.Instance.AddLayer("NewLayer");

12. InsertScriptToGameObject

public void InsertScriptToGameObject(GameObject gameObject, string script)

Inserts a custom Lua script to a game object.

  • gameObject: The game object to which the script should be attached.
  • script: The Lua script to be executed.

Example:

string customScript = "print('Custom script executed!')";
ModLoader.Instance.InsertScriptToGameObject(gameObject, customScript);
